Básico

1.- Fundamentos de programaciòn

  • Conceptos básicos de programación: variables, tipos de datos, operadores, estructuras de control de flujo, funciones, entre otros.
  • Introducción a Python: instalación, ejecución de scripts, uso de la consola interactiva.
  • Ejercicios básicos de programación en Python.

2.- Desarrollo web con HTML y CSS

  • Introducción a HTML y CSS: estructura de un documento HTML, selección de elementos en CSS, diseño de páginas web básicas.
  • Ejercicios prácticos de desarrollo web.

Programación orientada a Objetos

  • Conceptos básicos de programación orientada a objetos: clases, objetos, métodos, herencia, polimorfismo, encapsulamiento.
  • Implementación de programación orientada a objetos en Python.
  • Ejercicios prácticos de programación orientada a objetos.

Bases de Datos

  • Introducción a bases de datos: tipos de bases de datos, modelos de datos, SQL básico.
  • Conexión a bases de datos desde Python utilizando el módulo "sqlite3".
  • Ejercicios prácticos de manejo de bases de datos desde Python.

Intermedio

Desarrollo web con JavaScript

  • Introducción a JavaScript: sintaxis básica, tipos de datos, estructuras de control de flujo, funciones, eventos.
  • Manipulación del DOM con JavaScript.
  • Ejercicios prácticos de desarrollo web con JavaScript.

Desarrollo web con Flask

  • Introducción al framework Flask: instalación, estructura básica de una aplicación Flask, enrutamiento, plantillas Jinja2, formularios.
  • Creación de una aplicación web básica utilizando Flask.
  • Ejercicios prácticos de desarrollo web con Flask.

Desarrollo de APIs

  • Introducción al desarrollo de APIs: conceptos básicos, protocolo HTTP, métodos GET y POST, formato JSON.
  • Creación de una API RESTful básica utilizando Flask.
  • Ejercicios prácticos de desarrollo de APIs.

Avanzado

Desarrollo de aplicaciones escalables

  • Introducción a conceptos de arquitectura de software: patrones de diseño, principios SOLID, modularidad, escalabilidad.
  • Diseño e implementación de una aplicación escalable utilizando Flask y MongoDB.
  • Ejercicios prácticos de desarrollo de aplicaciones escalables.

Desarrollo de aplicaciones móviles

  • Introducción al desarrollo de aplicaciones móviles: conceptos básicos, arquitectura de aplicaciones móviles, desarrollo de interfaces de usuario.
  • Desarrollo de una aplicación móvil básica utilizando Python y Kivy.
  • Ejercicios prácticos de desarrollo de aplicaciones móviles.
